Multiple Linear Regression

What is Multiple Linear Regression?

Multiple Linear Regression हा Supervised Machine Learning Algorithm आहे. यात एका output साठी multiple input variables वापरले जातात.

Simple Linear Regression → 1 Input Variable

Multiple Linear Regression → Multiple Input Variables

Real Life Example

Student च्या marks prediction साठी खालील multiple factors वापरले जातात:

  • Study Hours
  • Attendance
  • Assignments Score

Sample Dataset

Study Hours Attendance % Assignment Score Final Marks
2 70 60 55
4 80 75 70
5 90 88 85
6 92 95 93

Formula

Y = b0 + b1X1 + b2X2 + b3X3
  • Y → Final Output
  • X1 → Study Hours
  • X2 → Attendance
  • X3 → Assignment Score
  • b0 → Intercept
  • b1,b2,b3 → Coefficients

Step-by-Step Working

Step 1: Multiple input variables collect करा
Step 2: Output variable identify करा
Step 3: Model relationship learn करतो
Step 4: Best fit equation तयार होते
Step 5: Future prediction होते

Formula Calculation Example

Y = 10 + 2X1 + 0.5X2 + 1.5X3
X1 = 5 Hours
X2 = 80 Attendance
X3 = 70 Assignment Score
Y = 10 + 2(5) + 0.5(80) + 1.5(70)
Y = 10 + 10 + 40 + 105
Final Prediction = 165
