What is Simple Linear Regression?

Simple Linear Regression हा एक Supervised Machine Learning Algorithm आहे. हा algorithm दोन variables मधील relationship शोधतो.

Example: Study Hours वाढले तर Marks वाढतात.

Real Life Example

Study Hours (X) Marks (Y)
1 35
2 45
3 52
4 65
5 75
X = Independent Variable
Y = Dependent Variable

Simple Linear Regression Formula

Y = mX + c
  • Y → Predicted Output
  • X → Input Variable
  • m → Slope
  • c → Intercept
Slope आणि Intercept हे Linear Regression चे सर्वात important parts आहेत.

What is Slope (m)?

Slope म्हणजे X मध्ये change झाला तर Y किती बदलतो हे दाखवतो.

m = Change in Y / Change in X
Study Hours Marks
2 45
4 65
Change in Y = 65 - 45 = 20
Change in X = 4 - 2 = 2

m = 20 / 2 = 10

Meaning: प्रत्येक 1 hour study वाढल्यावर marks 10 ने वाढतात.

What is Intercept (c)?

Intercept म्हणजे X = 0 असताना Y ची value काय असेल.

Example:
जर student ने 0 hours study केले तरी model काही base marks दाखवू शकतो.
Y = 10X + 25
येथे c = 25 आहे.

म्हणजे X = 0 असताना predicted marks = 25

Step-by-Step Formula Working Animation

Step 1: Formula घेतो → Y = mX + c
Step 2: m = 10 आणि c = 25 घेतो
Step 3: X = 5 hours study
Step 4: Y = 10(5) + 25
Step 5: Y = 50 + 25
Final Answer: Y = 75 Marks
Model predict करतो की student ला 75 marks मिळू शकतात.
