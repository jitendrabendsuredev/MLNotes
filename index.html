What is Machine Learning?

Machine Learning म्हणजे computer ला manually programming न करता data मधून शिकवणे. Computer patterns शोधतो आणि future prediction करतो.

Example: YouTube Recommendations, Netflix Suggestions, Spam Email Detection

Types of Machine Learning

Type Description Example
Supervised Learning Input + Output data available असतो House Price Prediction
Unsupervised Learning Only input data available असतो Customer Segmentation
Reinforcement Learning Reward & Penalty वापरून model शिकतो Game Playing AI

Supervised Learning Algorithms

Algorithm Use
Linear Regression Continuous value prediction
Logistic Regression Yes/No prediction
Decision Tree Rule based prediction
Random Forest Multiple trees for better accuracy
KNN (K-Nearest Neighbors) Nearest data points based prediction
Support Vector Machine (SVM) Classification problems
Naive Bayes Spam detection, text classification
Supervised Learning मध्ये output column available असतो.

Unsupervised Learning Algorithms

Algorithm Use
K-Means Clustering Groups तयार करणे
Hierarchical Clustering Tree structure grouping
PCA Dimensionality Reduction
Association Rule Learning Market Basket Analysis
Unsupervised Learning मध्ये output column नसतो.

Reinforcement Learning Algorithms

Algorithm Use
Q-Learning AI Agents Training
Deep Q Network (DQN) Game AI
Policy Gradient Advanced decision systems
Reinforcement Learning मध्ये agent reward घेऊन शिकतो.

Detailed Study: Supervised Learning

Supervised Learning मध्ये model ला Input आणि Output दोन्ही data दिला जातो. म्हणजेच model ला आधीच correct answers माहित असतात. त्यावरून model future prediction शिकतो.

Formula: Input Data + Correct Output = Supervised Learning

Types of Supervised Learning

Type Purpose Example
Regression Number Prediction Salary Prediction
Classification Category Prediction Spam or Not Spam

Example 1: Student Marks Prediction (Regression)

Student किती hours study करतो त्यावरून marks predict करणे.

Study Hours Marks
1 35
2 40
3 50
4 60
5 72
येथे output numeric आहे म्हणून हा Regression problem आहे.

Example 2: Email Spam Detection (Classification)

Email spam आहे की नाही हे predict करणे.

Email Text Output
Win ₹10,000 now! Spam
Meeting at 5 PM Not Spam
Claim your free gift Spam
Project deadline tomorrow Not Spam
येथे output category आहे म्हणून हा Classification problem आहे.
